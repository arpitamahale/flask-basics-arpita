Academic Report

This page demonstrates passing a dictionary and looping through it using Jinja2.

{% for subject, score in grades.items() %} {% endfor %}
Subject Grade
{{ subject }} {{ score }}

How It Works

In app.py, we pass a dictionary to the template:

@app.route('/grades') def grades(): subjects_and_grades = { 'Mathematics': 'A', 'Computer Science': 'A+', 'Physics': 'B' } return render_template('grades.html', grades=subjects_and_grades)

Jinja2 Dictionary Looping

To display the dictionary, we use the .items() method inside a for-loop:

{% for subject, score in grades.items() %} <tr> <td>{{ subject }}</td> <td>{{ score }}</td> </tr> {% endfor %}
