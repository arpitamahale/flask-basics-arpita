About the To-Do List App

This is a Flask-based To-Do List web application designed to help users manage tasks efficiently with a clean and modern user interface.

What is Flask?

Flask is a lightweight and flexible Python web framework used to build web applications quickly and efficiently. It follows the Model-View-Template (MVT) architecture and provides full control to developers without unnecessary complexity.

Technologies Used

  • Flask (Python): Backend framework for routing, logic, and data handling.
  • HTML: Used to create the structure of web pages.
  • CSS: Used to style the application and create a professional UI.

Key Features

  • Create and view tasks
  • Task priority management (High, Medium, Low)
  • Task status tracking (Completed, In Progress, Pending)
  • Clean and responsive user interface
  • Light and Dark mode theme switcher

Project Purpose

This project is developed as a learning and practice application to understand Flask fundamentals, routing, templates, and frontend integration. It follows industry-style design principles and can be extended with databases and authentication.
